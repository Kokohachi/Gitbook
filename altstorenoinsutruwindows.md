# 🔷 AltStoreのインストール(Windows)

## **AltStore導入の準備:**

1. **iTunes と iCloud**をAppleから直接ダウンロード・インストール

{% hint style="danger" %}
Microsoft Storeからはインストールしないでください。

既にある場合一旦アンインストールしてください。
{% endhint %}

{% hint style="info" %}
**iTunes のダウンロード:**

Windows 64ビット: [https://www.apple.com/itunes/download/win64](https://www.apple.com/itunes/download/win64)

Windows 32ビット: [https://www.apple.com/itunes/download/win32](https://www.apple.com/itunes/download/win32)
{% endhint %}

{% hint style="info" %}
**iCloudのダウンロード:**

[https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe)
{% endhint %}

## **AltStoreのインストール:**

1. [altstore.io](https://altstore.io)からWindows用のファイルをダウンロードする
2. ダウンロードしたzipファイルを展開し、中の２つのファイルどちらかを実行
3. 検索タブで`AltServer`と検索し、管理者権限で実行。左下に四角いマークが出ます。

4\. ご自身のiOSデバイスをロック解除し、まだしていなければ信頼してください。

5\. iTunesを開き、Wi-Fi同期をONにしてください。

6\. 右下のAltServerアイコン　◇　をクリックして、Install AltStore →デバイス名　を選択。

7\. Apple IDとパスワードを入力してください。このApple IDによってAltStoreやそれで入れたアプリが動くようになります。

{% hint style="success" %}
あなたのApple IDは**Appleにだけ**送信されます。\
筆者もためしていますが、安全です。
{% endhint %}

8\. 数秒待つと、デバイスにAltStoreがインストールされるはずです。

{% hint style="info" %}
出てこない場合は再起動してみてください。
{% endhint %}

9\. iOS デバイスにおいて、 設定 -> 一般 -> VPNとデバイス管理　をタップし、 “ \[Apple ID]を信頼する” をタップ。

![Image courtesy of @LouisAnslow](https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FAfe8qEztjcTjsjjaMBY2%2Fuploads%2F1NUuIKBgS0adjhFhOqVt%2F1-CwxM2iiAN0kLSWGuRQcCHQ-2%20\(dragged\).jpg?alt=media\&token=f94229f1-89be-4e25-854d-d13736f64ca3)

{% hint style="info" %}
VPNとデバイス管理はiOSバージョンによって表記が違いますが、「一般」の下の方の項目にそのようなものが必ずあるはずです。お手数おかけしますがお探しください。
{% endhint %}

**これでAltStoreが使用できます！**
